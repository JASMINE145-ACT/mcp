背后到底发生了什么？

Part 01从"氛围编程"到"智能体工程"

智谱将 GLM-5 定位为推动编程范式转变的旗舰模型——从 VibeCoding（氛围编程） 跨越至 Agentic Engineering（智能体工程）。

这不是措辞升级，而是底层逻辑的重构。

传统 AI 编程助手的工作模式是：你描述需求，模型生成代码片段，你复制粘贴，再手动调试。模型提供"灵感"，人仍然在开车——这就是"氛围编程"。

智能体工程则要求模型理解整个软件工程任务，自主拆解需求、调用工具链、编写测试、迭代修复，完成端到端的软件开发流程。这需要持续推理、长程规划与自我纠错能力，而不仅仅是代码补全。

GLM-5 的目标，是成为那个独立坐在 IDE 前写完整个项目的工程师。

在 GLM-5 发布后，智谱宣布将 GLM Coding Plan 套餐涨价：中国区涨价 30%，海外版涨价超 100%——成为国内首家因模型能力增强而主动提价的 AI 企业。涨价后，新套餐上线即售罄。

Part 02四大技术创新，744B参数的秘密

GLM-5 的规模令人侧目：744B 总参数，每次推理激活约 40B。依赖 MoE（混合专家）架构，GLM-5 技术报告披露了四项核心创新：

第一，DSA 稀疏注意力机制（DeepSeek Sparse Attention）。根据 Token 重要性动态分配注意力资源，大幅降低训练与推理成本，同时保持长上下文能力无损。训练 Token 规模扩展至 28.5万亿。

第二，异步强化学习基础设施。将生成过程与训练过程解耦，大幅提升后训练迭代效率，使模型能从复杂、长程交互中高效学习。

第三，异步 Agent RL 算法。专为复杂智能体任务设计，提升模型在多轮、多工具场景下的决策质量与完成率。

第四，ZCube 下一代推理架构。千卡生产集群落地，推理吞吐提升 15%，硬件成本降低 33%，支持 400 tokens/s 高速输出，适配 AI 编程、实时交互等低延迟场景。

成本方面，GLM-5 每任务费用约为 0.14 美元，而 Claude Opus 4.6 为 6.39 美元。完成同等智能体任务，GLM-5 的成本仅为对手的 1/45。

Part 05为什么偏偏是智谱

中国有几十家大模型公司，为什么智谱能杀出来？三条护城河值得关注。

护城河一：技术路线押注正确。 当大多数竞争对手还在通用对话模型上内卷，智谱从 GLM-4.5 时代起就明确押注 Agentic + Reasoning + Coding（ARC） 三条赛道。GLM-5 的所有创新都指向同一个目标：让模型独立完成真实世界的软件工程任务。押注编程，而不是打榜，这个判断在 2026 年被证明是正确的。

护城河二：国产算力深度适配。 GLM-5 完整适配华为昇腾、摩尔线程、寒武纪、昆仑、燧原、壁仞、海光等七款国产芯片。这既是国产化要求，也是绕开英伟达禁令的战略保险。657名研发人员（占总员工74%）中，相当一部分专注于芯片适配与推理优化，形成竞争对手难以复制的底层能力。

护城河三：先发的全球开发者生态。 24.2万付费开发者、218个国家覆盖，以及 Windsurf、Cerebras、Vercel 等海外顶级平台集成，构成了强力的生态飞轮。开发者用 GLM 构建产品，产品收入反哺模型训练，训练结果进一步吸引开发者——这个循环一旦建立，就很难被打破。

截至2026年3月，智谱平台注册用户已超400万，服务覆盖全球218个国家和地区。2026年6月，智谱将正式纳入恒生科技指数，初始权重约0.53%，预计带来12.5至17.5亿美元被动资金流入。

股价是结果，不是原因。

从116港元到725港元，市场在定价的，是一条被真实收入验证的技术路线，是一支在最难打的编程赛道建立了全球开源第一地位的团队，以及一个在"价格战"中唯一成功涨价的商业模式。

智谱凭什么？凭真实的工程师需求，凭逆势的定价权，凭七年没有动摇的技术押注。
